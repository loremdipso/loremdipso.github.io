Grabs the content of a given recipe and makes it as useful as possible, for free! Current features include:

- [x] Recipe syntax highlighting
- [x] Unit conversion
- [x] Shopping list support
- [x] Local storage
- [x] Import/export

You can even install it as a PWA 😎
